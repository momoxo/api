class XoopsConfigHandler

XOOPS configuration handling class.

This class acts as an interface for handling general configurations of XOOPS
and its modules.

Properties

object $_cHandler holds reference to config item handler(DAO) class
$_oHandler holds reference to config option handler(DAO) class
array $_cachedConfigs holds an array of cached references to config value arrays, indexed on module id and category id
